Bitcoin mining is a process that achieves two-goals for the network.

  1. It keeps a record of all the transactions going across the network, ensuring that they are legitimate transactions.
  2. It is a discovery process by which new bitcoins are created. Since there is no central authority to print money (as is the case with national currencies) the Bitcoin network rewards the miners who are contributing their computing power to secure the network.

Technical Aspects of Mining

A computer connects to the Bitcoin network and performs a SHA256 hash verification processes in order to validate bitcoin transactions and provide the requisite security for the public ledger of the Bitcoin network.

A hash is a computer operation that is relatively easy to perform, taking data (for instance from all the transactions during a set period of time) and creating a seemingly random number. However hashes are almost impossible to back-compute. Once a miner finds a valid block, by completing the hash function then it becomes time-stamped and associated with the previous blocks (hence the term blockchain). Any attempted to change even one transaction would cause the hash to change.

The bitcoin network runs of the longest chain. This is the chain with the most complete work. In the event that there was a dispute about transactions then each miner will have to choose which chain to adopt.

There are broadly two ways to get started with mining

  • Mining by purchasing your own hardware.
  • Cloud mining

To get started hardware mining you need

  1. Hardware
  2. Software
  3. An internet connection.
  4. Source of electricity.

4 years ago, when the bitcoin network was much younger it was possible to mine with the CPUs from computers, even a laptop would work. But, as more miners connected to the network the difficulty of finding a block (and earning rewarded) increased.

Today, you need to buy specialized hardware designed specifically for bitcoin mining. Most of these manufacturers are based in China and produce ASIC (application-specific integrated circuit) computers, with the software for mining pre-installed. View main product examples

The other option if one does not wish to invest in hardware is to buy a cloud mining contract. These contracts are sold by data centers that already own mining hardware and will sell contracts to rent out for a specified amount of time. For example www.genesis-mining.com.

Another thing to note is that the difficulty on the Bitcoin network is so high that it is impossible to solo mine. You will need to connect to a mining pool, where you share your hashing power with a group of other miners and then share the rewards collectively. An example of a mining pool

At this point in time the profitability is dependent on several factors, including

  1. Cost of hardware
  2. Cost of electricity
  3. Bitcoin price
  4. Mining difficulty

The mining operations that will be most profitable will be those that can obtain electricity at low cost and scale large enough so that the return on investment covers the investment cost in the shortest amount of time.

To estimate the profitability, you can use a mining profitability calculator for example http://www.bitcoinx.com/profit/

At the minimum you need an

  1. ASICS computer that compute the Sha256 algorithim.
  2. Software that will connect you through a stratum proxy to a
  3. Mining pool and you will need
  4. A reliable electricity source and a
  5. Reliable internet connection.
